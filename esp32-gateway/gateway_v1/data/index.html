Web Interface Access

Protect the gateway web interface with a username and password. By default, the interface is open — no login is required. Once credentials are set, all users must log in to access the dashboard.

Once credentials are set, there is no way to recover a forgotten password. You must manually factory reset the Gateway via the hardware push button to regain access to the web interface.
